# Habitris 도메인

## 용어
- **습관 완료** → 해당 습관에 대응하는 **테트리스 블록이 자동으로 쌓임**
- **직접 조작** → 사용자가 블록 배치·회전·이동
- **줄 완성** → 한 줄이 꽉 차면 그 줄 제거 후 **Castle 페이지에 기록**
- **Castle** → 완성된 줄들이 성처럼 쌓이는 별도 화면

## 플로우
1. 습관 목록에서 완료 체크 → 블록 자동 낙하/쌓기
2. 또는 사용자가 테트리스 필드에서 블록 직접 조작
3. 한 줄이 완성되면 줄 제거 + Castle에 해당 줄 추가
4. Castle 화면에서 완성된 줄 히스토리 확인

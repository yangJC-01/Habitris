# Habitris 기획서

## 1. 개요

**Habitris**는 습관 추적을 **테트리스** 형태로 표현하는 미니멀한 습관 트래커 애플리케이션이다.  
습관 완료 시 블록이 쌓이고, 한 줄이 완성되면 Castle 페이지에 기록되는 구조이다.

---

## 2. 플랫폼

| 단계 | 플랫폼 | 비고 |
|------|--------|------|
| 1차 | **PC 앱** | 데스크톱용 먼저 개발 |
| 2차 | **모바일 앱** | Play Store 배포 대상 |

---

## 3. 수익 모델

- **다운로드**: 무료
- **광고**: 수익화용 광고 노출
  - 사용성에 크게 방해되지 않는 수준으로 배치 (예: 완료 후, 화면 전환 시 등)

---

## 4. 핵심 기능

### 4.1 테트리스형 습관 트래킹

- 습관을 **완료**하면 → 해당 습관에 대응하는 **테트리스 블록이 자동으로 쌓임**
- **직접 조작**도 가능: 사용자가 블록을 직접 배치·회전·이동할 수 있음

### 4.2 줄 완성 → Castle 연동

- 테트리스에서 **한 줄이 완성**되면:
  - 해당 줄은 **제거**되고
  - **Castle 페이지**에 그 줄이 **쌓여서 기록**됨
- Castle: 완성된 줄들이 모여 성(城)처럼 쌓이는 별도 화면

---

## 5. 화면/구성 (요약)

| 화면 | 설명 |
|------|------|
| **습관·테트리스** | 습관 목록 + 테트리스 필드 (자동/수동 블록 쌓기) |
| **Castle** | 완성된 줄들이 쌓인 결과를 보여주는 페이지 |

---

## 6. 정리

- **컨셉**: 습관 완료 = 테트리스 블록, 줄 완성 = Castle에 기록
- **플랫폼**: PC 앱 → 모바일(Play Store)
- **수익**: 무료 + 사용성을 해치지 않는 광고
- **조작**: 자동 쌓기 + 직접 조작 모두 지원
